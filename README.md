# 감정 감정사 (Gamjeong Appraiser)

> "안녕하세요." 썼는데 너무 딱딱한가?
>
> "확인 부탁드립니다." 썼는데 싸가지 없어 보이나?
>
> "감사합니다 😊" 붙였는데 너무 아부 같나?
>
> 메시지 하나 보내는데 10분 고민해본 사람들을 위한 서비스입니다.

감정 감정사는 AI가 문장의 톤을 분석하고,
상황에 맞는 더 나은 표현을 제안해주는 커뮤니케이션 도우미입니다.

## 프로젝트 소개

직장이나 일상에서 메시지를 보내기 전,
내 문장이 상대방에게 어떻게 전달될지 고민될 때가 있습니다.

감정 감정사는 문장의 톤을 객관적으로 분석하여
따뜻함, 격식, 직접성, 자신감을 점수화하고,
상황에 맞는 개선된 문장을 제안합니다.

## 주요 기능

- 상대방 유형(상사, 고객, 팀원, 파트너)에 따른 문장 분석
- 따뜻함 · 격식 · 직접성 · 자신감 4가지 지표 시각화
- 프로페셔널 / 따뜻함 / 간결함 버전의 문장 개선안 제공
- 카카오 소셜 로그인 및 분석 내역 저장
